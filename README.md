### Como rodar o projeto localmente

Após clonar o projeto instale as dependências e crie o arquivo .env na base do projeto. 

- 1
```javascript
npm install
```

- 2
```javascript
touch .env
```

No arquivo .env insira as informações necessárias.

```
DRAMA_NEWS_API_URL=

DRAMA_NEWS_API_KEY=

DRAMA_NEWS_API_TIMEOUT=

```

Agora só precisa rodar o comando:

```
npm run dev
```
